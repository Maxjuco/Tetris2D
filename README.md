# Tetris2D

<B>Description :</B><br>
A clone of Tetris developed on Unity
<br><br><br>

<B>Installation :</B>
- Download the compressed files Build_2.0 (lastest version)
- extract in a folder
- launch the application by double clicking on tetris2D.exe
</br></br></br>

<B>How to play :</B>
- classic tetris control with the key arrows
</br></br></br>
